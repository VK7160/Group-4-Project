Abstract

The motive behind Water tank managment project is to save maximum amount of water while filling and removing tank as well as this project will provide smart autonomous service of water restoring. This project is an IoT based project so, it is an entirely new level up from the manual human checks.

Internet of things (IoT)

First of all, we used HTML to markup a webpage and CSS to style a webpage. Then our aim is to focus on displaying data for IoT application. Our next step is to build web server using node.js that will be responsible for back-end routing between pages. The service API should be done along with HTTP requests and implementing backend API. As testing allows us to write more reliable and predictable code we test with jest. Than we will generate API documentation in 'js' file to remeber the functioning of each code line. Moving further, we will Setting up MQTT for handling inbetween communications. In further stage, we secure a web server with HTTPS utilizing Helmet middleware. In end, we will use Heroku for deploying our application.
